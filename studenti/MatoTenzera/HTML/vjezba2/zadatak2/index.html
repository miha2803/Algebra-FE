Uvod u HTML

HTML je jezik za obiljezavanje koji koristimo za izradu web stranica.
Njime definiramo strukturu i sadrzaj web stranica.

Povijest HTML-a

HTML je nastao pocetkom 1990-ih godina.

Uređivanje teksta

Tekst je najcesci sadrzaj web stranica.
Da bi dobio odgovarajuću ulogu na stranici i bio prikazan na odredjeni način uređujemo ga HTML-om.

HTML atributi

Unutar pocetne oznaka elemenata, nakon naziva elementa, po potrebi pisemo i atribute.
Atributi internetskim preglednicima daju dodatne informacije o elementu.
