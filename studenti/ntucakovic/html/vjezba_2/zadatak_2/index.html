Uvod u HTML

HTML je kratica za HyperText Markup Language, što znači prezentacijski jezik za izradu web stranica.
Hipertekst dokument stvara se pomoću HTML jezika.

Povijest HTML-a

HTML je nastao početkom 1990.ih godina.

Uređivanje teksta

Tekst je najčešći sadržaj web stranica.
Da bi dobio odgovrarajuću ulogu na stranici i bio prikazan na određeni način uređujemo ga HTML-om.

HTML atributi

Unutar početne oznake elemenata po potrebi pišemo i atribute.
Antibuti internetskim preglednicima daju dodatne informacije o elementima. o samom atributu ovisi o kojim informacijama je riječ.
Atribut se sastoji od imena i vrijednosti atributa.
