Uvod u HTML

HTMLje jezik za obiljezavanje koji koristimo za izradu web stranica. Nime definiramo strukturu i sadrzaj web stranica.

Povijest HTMLa

HTML je nastao početkom 1990-ih godina.

Uredivanje teksa

Tekst je najčešći sadržaj web stranica. Da bi dobio odgovarajuću ulogu na
stranici i bio prikazan na određeni način uređujemo ga HTML-om.

HTML atributi

Unutar početne oznaka elemenata, nakon naziva elementa, po potrebi pišemo i atribute. Atributi internetskim preglednicima daju dodatne informacije o elementu. O samom atributu ovisi o kojim informacijama je riječ. Atribut se sastoji od imena i vrijednosti atributa.
